**1. 这个工具主要能做什么？**
**A:** "Nano Banana Flow" 是一个 Chrome 浏览器插件，专为 Google Gemini 设计。它可以让您：
*   🚀 **批量生图**：一次性输入多个提示词，全自动逐个生成。

*   📥 **自动保存**：自动将生成的图片下载保存到本地，彻底解放双手。

## 🔥 第二部分：高频痛点 Top 3

**3. 插件无法加载，或者一直显示“请先打开 Gemini 页面”？**
**A:** 这是最常见的问题。
1.  **检查网址**：必须在 [https://gemini.google.com/app](https://gemini.google.com/app) 页面使用。
2.  **刷新页面**：有时候页面元素（如输入框）还没完全加载出来，刷新一下通常能解决。
3.  **检查节点**：如果您的网络节点自动跳转到了特定区域的 Gemini（网址变动），插件可能无法识别。尝试切换到美国节点。

**4. 为什么下载的文件是 `.html` 网页文件，而不是图片？**
**A:** 这是浏览器拦截了自动下载。请按以下步骤设置：
1.  在地址栏最左侧点击“设置”图标（锁形或调节图标）。
2.  进入 **“网站设置 (Site Settings)”**。
3.  找到 **“自动下载 (Automatic downloads)”**，改为 **“允许 (Allow)”**。
4.  刷新页面重试。

**5. 如何让图片自动匹配到提示词？（命名规则是什么？）**
**A:** 这就是所谓的**参考图（垫图）**功能，**特别适合电商场景（如批量给商品图换背景）**。
规则非常灵活，**只看文件名开头的数字**：
*   **支持格式**：`1_cat.jpg`, `1-photo.png`, `01.jpeg`, `2 狗.webp` 等。
*   **匹配逻辑**：数字 `1` 对应第 1 行提示词，`2` 对应第 2 行。
*   **操作**：点击插件上的 "🖼️ 导入图片" 按钮即可。

## 🛠️ 第三部分：进阶使用指南

**6. 如何正确输入提示词？**
**A:** **一行一个**。不要把所有内容写在同一行。
```
一只猫
一只狗
一只鸟
```
工具会按行顺序执行。

**7. 过程中可以停止吗？**
**A:** 可以。点击红色的“停止”按钮。注意：已发送给 Gemini 的指令无法撤回，需等待该张生成完毕后才会完全停止。

**8. 生成的图片保存在哪里？文件名是怎样的？**
**A:**
*   **位置**：浏览器的默认“下载”文件夹。如果在插件里填了子文件夹名，会由浏览器自动归档。
*   **文件名**：默认按顺序命名，如 `page1.png`, `page2.png` 等。

**9. 是否支持“图生图”？**
**A:** 是的，v1.2.0 版本已全面支持。使用上述的“导入图片”功能即可。

## 🔧 第四部分：疑难杂症排除

**10. Gemini 界面是繁体中文，插件不工作？**
**A:** 请确保使用 **v1.1.1** 及以上版本。如果仍有问题，临时将 Google 语言设置为“英文”或“简体中文”可立即解决。

**11. 图片生成了但没有自动下载？**
**A:**
1.  检查浏览器是否允许插件下载（权限问题）。
2.  保存目录名不要包含特殊符号。
3.  网络超时（超过2分钟）可能导致下载失败。

**12. 为什么生成的数量不对（比如输入3个只出了2张）？**
**A:**
1.  **网络波动**：Gemini 响应超时，插件跳过了该任务。
2.  **内容审查**：提示词触犯了 Google 的安全策略，图片被拦截未生成。

**13. 侧边栏挡住了视线怎么办？**
**A:** 任务开始后，再次点击浏览器右上角的插件图标，侧边栏就会**收起**。这不会影响正在运行的任务。

## 📦 第五部分：安装教程

**14. 如何安装此插件？**
**A:**
1.  下载 zip 包并解压。
2.  Chrome 地址栏输入 `chrome://extensions/`。
3.  开启右上角“开发者模式”。
4.  点击“加载已解压的扩展程序”，选择解压后的文件夹。
