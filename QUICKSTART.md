# 🚀 快速开始指南

## 5分钟上手Gemini Auto PPT Generator

### 第1步：加载插件（2分钟）

1. 打开Chrome浏览器，访问：`chrome://extensions/`
2. 打开右上角的"**开发者模式**"开关
3. 点击"**加载已解压的扩展程序**"
4. 选择文件夹：`/Users/gt/AI_Code/2.Antigravity Build/4.Nano Banana Pro Plugging`
5. 看到插件卡片出现即成功！

### 第2步：打开Gemini（30秒）

访问：**https://gemini.google.com/app**

### 第3步：使用插件（2分钟）

1. 点击Chrome工具栏的插件图标 🥖
2. 在输入框中输入提示词（每行一个），例如：
   ```
   一只可爱的猫咪
   一只金毛犬
   一朵玫瑰花
   ```
3. 点击"**🎨 开始生成**"
4. 等待自动完成！

### 第4步：查看结果（30秒）

打开下载文件夹，查看生成的图片：
- `page1.png`
- `page2.png`
- `page3.png`

## 🎯 使用技巧

### 提示词格式
- ✅ 每行一个提示词
- ✅ 可以是中文或英文
- ✅ 建议简洁明确

### 批量生成
- 一次可以输入多个提示词
- 插件会自动逐个生成
- 进度条实时显示

### 文件管理
- 图片自动命名为 `page1.png`, `page2.png`...
- 保存在Chrome默认下载文件夹
- 如果文件名重复，Chrome会自动添加(1)、(2)等后缀

## ⚠️ 注意事项

1. **必须在Gemini页面使用**
   - 确保在 `https://gemini.google.com/app`
   - 必须已登录Google账号

2. **等待生成完成**
   - 每张图片生成需要时间
   - 不要关闭Gemini页面
   - 进度条会显示当前状态

3. **网络连接**
   - 需要稳定的网络连接
   - 图片生成依赖Gemini服务
